• Intro to ES6

Default Parameters

es5

                
                  function(x,y) {
                    var a = x || 1;
                    var b = y || 1;
                    return a+b;
                  }

Output:

es6

                
                  function(x=1,y=1) {
                    return x+y;
                  }

String Literals

es5

              
                var first = "Rupesh";
                var last = "B";
                var text = "My Name is" + " "+ first + " " + last;

Output:

es6

              
                var first = "Rupesh";
                var last = "B";
                var text = `My Name is ${first} ${last}`;
